Local Server Required

Because of browser security (CORS), web browsers can't load the xkcd_embeddings.json file directly using the file:// protocol. To view your search app, open your terminal in this folder and start a quick static server: 

python -m http.server 8000

Then, open http://localhost:8000 in your browser!
